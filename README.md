# HireOn – Blog Management System

HireOn is a Laravel-based Blog Management System developed for the JobYaari PHP/Laravel Developer Assessment. The platform allows users to explore the latest job updates, admit cards, results, internships, and recruitment notifications with AJAX-powered filtering and responsive UI.

# Features

## User Side
- View all blogs dynamically from database
- Responsive modern UI
- Blog detail page
- AJAX-based search functionality
- Filter blogs by category
- Filter blogs by date
- Mobile-friendly design
- Public blog access without login

## Admin Side
- Admin authentication system
- Create blog posts
- Edit existing blogs
- Delete blogs
- Upload blog images
- Manage blog content and categories

---

# AJAX Features

- Dynamic category filtering
- Date filtering without page reload
- Real-time search functionality using jQuery AJAX
